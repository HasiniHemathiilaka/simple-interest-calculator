# Simple Interest Calculator

This project provides a simple interest calculator.

## Formula
Simple Interest (SI) = (P × R × T) / 100

Where:
- P = Principal amount
- R = Annual interest rate (%)
- T = Time (years)

## Example
If P = 1000, R = 5, T = 2  
SI = (1000 × 5 × 2) / 100 = 100
